Adds a filter to WooCommerce orders to filter by order origin and displays total sales for the filtered origin.

== Description ==

Origin Filter for WooCommerce enhances your WooCommerce store by adding a filter on the orders page to filter orders by their origin. This allows you to easily sort and view orders based on their origin. Additionally, the plugin displays the total sales amount for the selected origin, helping you gain insights into sales performance by origin.

Key Features:
* Dropdown filter to select order origin on the WooCommerce orders page.
* Total sales calculation for the selected origin.
* Display total sales amount on the admin orders page.

The plugin caches origin data for better performance and validates nonce values to ensure security.

== Frequently Asked Questions ==

= How do I use this plugin? =

After activating the plugin, you will see a dropdown filter on the WooCommerce orders page allowing you to filter orders by origin. The plugin will also display the total sales for the selected origin.

= What if no orders match the selected origin? =

If there are no orders matching the selected origin, the plugin will display a notice indicating that no orders were found for the selected origin.
