Adds a filter to WooCommerce orders to filter by order Origin , Payments and displays total sales for the filtered origin.

== Description ==

Origin Filter for WooCommerce enhances your WooCommerce store by adding advanced filtering options on the orders page. You can filter orders based on both origin and payment gateway, making it easier to sort and view orders by these criteria. Additionally, the plugin calculates and displays the total sales for the selected filters, providing valuable insights into sales performance.

Key Features:
* Dropdown filter to select order origin and payment gateway on the WooCommerce orders page.
* Total sales calculation for the selected origin and payment gateway combination.
* Admin notice displaying the total sales amount, number of orders, and the filters used.
* Seamless integration where all filters work together smoothly.
* User-friendly interface for easy filtering without any conflicts.

The plugin caches origin data for better performance and validates nonce values to ensure security.

== Frequently Asked Questions ==

= How do I use this plugin? =
After activating the plugin, you will see a dropdown filter on the WooCommerce orders page that allows you to filter orders by origin and payment gateway. The plugin will also calculate and display the total sales for the selected origin and payment gateway combination.

= What if no orders match the selected filters? =
If no orders match the selected filters (origin or payment gateway), the plugin will display a notice indicating that no orders were found for the selected filters.

= Can I use multiple filters at once? =
Yes! All filters work seamlessly together, so you can filter orders by both origin and payment gateway simultaneously.

= What information will be shown in the admin notice? =
The admin notice will display a summary of the applied filters, including the selected origin, payment gateway, the number of orders that match, and the total sales amount for the filtered orders.

= What should I do if I encounter an issue with the plugin? =
If you encounter any issues or bugs, please check the plugin documentation for troubleshooting steps. You can also contact support with detailed information about the issue for assistance.

= Does the plugin support all payment gateways? =
The plugin displays all payment gateways that have been used in the orders. If a payment gateway does not appear in the filter, it means that it has not been used in any orders. Make sure the payment gateway is properly configured in your WooCommerce settings.
